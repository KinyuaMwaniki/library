## Installation

- git clone https://github.com/KinyuaMwaniki/wiz-erp-felix.git-
- cd wiz-erp-felix\
- composer install --ignore-platform-reqs
- npm install
- npm run dev
- cp .env.example .env
- php artisan key:generate
