## Installation

- git clone https://github.com/KinyuaMwaniki/tracking.git
- cd tracking
- composer install --ignore-platform-reqs
- npm install
- npm run dev
- cp .env.example .env
- php artisan key:generate
- Create db
- php artisan migrate --seed
